Da console Influx:

	- Creare nuovo DB:
		- CREATE DATABASE nome_db
	
	- Mostra i DB esistenti:
		- SHOW DATABASES
		
	- Seleziona il DB
		- USE nome_db
		- Da ora in poi tutte le query sono per questo db
